# Codex Now Generally Available

- **原文链接**: [Codex now generally available](https://openai.com/index/codex-now-generally-available/)
- **发布日期**: 2025-10-06（与 DevDay 2025 同日）
- **标签**: #Codex-GA #Slack集成 #Codex-SDK #企业控制 #语音编码

## 核心观点

Codex 从研究预览版正式进入 GA（Generally Available）。自 2025 年 5 月发布云 Agent 以来，使用量增长了 10 倍。GA 版本新增 Slack 集成、Codex SDK 和企业管理工具。

## 关键概念

### 使用量增长
- 自 2025 年 8 月初以来使用量增长 10 倍
- ChatGPT 拥有超过 8 亿周活跃用户

### 新增功能

1. **Slack 集成**
   - 在 Slack 频道或线程中直接调用 Codex
   - 团队协作编码

2. **Codex SDK**
   - 将 AI 辅助工作流嵌入自定义应用和 CI/CD 管道
   - 目前支持 TypeScript，更多语言即将推出
   - GitHub Action 可用于 CI/CD

3. **企业管理工具**
   - 管理环境、配置和使用分析
   - 代码审查质量跟踪
   - 环境控制强制执行

4. **语音编码演示**
   - 现场演示了语音控制编码（如舞台灯光控制）

## 实践启示

- Codex GA 标志着编码 Agent 从实验走向生产
- 10 倍使用量增长验证了市场需求
- 企业级功能（管理工具、环境控制）是大规模采纳的关键
- Slack 集成将编码 Agent 带入团队协作场景
