# L0 预备阶段：编程基本功

> **阶段定位**：零基础学员的第一站，建立编程思维和工具使用习惯。
>
> **核心教学理念**：不做PPT讲师，做项目教练。每个技术点按照"感性认识→动手搭建→原理深挖→实战项目"四步走。

---

## 阶段概述

| 项目 | 说明 |
|------|------|
| 周期 | 8周（含2周项目实战） |
| 总课时 | 160小时面授 |
| 项目数 | 3个综合项目 |
| 目标角色 | 编程入门（为大数据学习打基础） |

---

## 阶段目标

学员结业时能够：

1. **Python编码**：独立编写数据处理脚本，掌握面向对象编程，熟练使用Python标准库
2. **SQL能力**：熟练编写中等复杂的SQL查询（JOIN、子查询、窗口函数）
3. **Linux操作**：在纯命令行环境下管理Linux服务器，使用awk/sed/grep处理文本
4. **Git版本管理**：使用Git进行日常代码版本管理，理解分支工作流
5. **项目交付**：独立完成3个完整项目，从需求分析到代码实现到报告输出
6. **Docker与容器化**：理解容器化基本概念，能使用Docker拉取镜像、运行容器、构建简单镜像
