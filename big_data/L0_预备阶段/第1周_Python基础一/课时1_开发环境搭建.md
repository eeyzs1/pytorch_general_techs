### 2.1 什么是Python？（15分钟）

**Python是一门"胶水语言"**，在数据工程领域广泛应用：

- **数据采集**：爬虫（Scrapy）、API调用
- **数据处理**：Pandas、NumPy
- **数据管道**：Airflow DAG定义、ETL脚本
- **机器学习**：PyTorch、scikit-learn
- **DevOps**：自动化运维脚本

**Python 3.x 核心特点**：
- 解释型语言：不需要编译，直接运行
- 动态类型：变量无需声明类型
- 缩进即代码块：强制代码整洁
- 丰富的标准库和第三方生态

### 2.2 Anaconda发行版安装（20分钟）

**为什么要用Anaconda而不是纯Python？**

| 对比维度 | 纯Python | Anaconda |
|----------|----------|----------|
| 包管理器 | pip | conda（更强大） |
| 科学计算包 | 需手动安装 | 预装NumPy/Pandas等 |
| 虚拟环境 | venv | conda env（更易用） |
| 依赖解决 | 可能冲突 | conda自动解决 |

**安装步骤（Windows）**：

1. 访问 https://www.anaconda.com/download
2. 下载Python 3.10+版本的安装包
3. 安装时勾选"Add Anaconda to PATH"
4. 验证安装：

```bash
conda --version
python --version
```

**配置国内镜像源（加速下载）**：

```bash
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --set show_channel_urls yes
```

### 2.3 虚拟环境管理（20分钟）

**为什么需要虚拟环境？**

不同项目可能需要不同版本的依赖包。例如：
- 项目A需要Pandas 1.3
- 项目B需要Pandas 2.0

虚拟环境让每个项目拥有独立的Python环境，互不干扰。

**核心命令**：

```bash
# 创建虚拟环境（指定Python版本）
conda create -n bigdata python=3.10

# 激活虚拟环境
conda activate bigdata

# 查看所有虚拟环境
conda env list

# 在环境中安装包
conda install numpy pandas matplotlib

# 导出环境配置
conda env export > environment.yml

# 从配置创建环境
conda env create -f environment.yml

# 退出虚拟环境
conda deactivate

# 删除虚拟环境
conda env remove -n bigdata
```

### 2.5 第一个Python程序（15分钟）

**方式一：交互式解释器**

```bash
python
>>> print("Hello, Big Data!")
Hello, Big Data!
>>> import sys
>>> print(f"Python版本: {sys.version}")
>>> print(f"运行平台: {sys.platform}")
>>> exit()
```

**方式二：脚本文件**

创建 `hello_bigdata.py`：

```python
import sys
import platform
from datetime import datetime

def main():
    print("=" * 50)
    print("       大数据学习之旅——环境验证")
    print("=" * 50)
    print(f"Python版本: {sys.version}")
    print(f"运行平台: {sys.platform}")
    print(f"操作系统: {platform.system()} {platform.release()}")
    print(f"处理器: {platform.processor()}")
    print(f"当前时间: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
    print("=" * 50)

if __name__ == "__main__":
    main()
```

运行：

```bash
python hello_bigdata.py
```

### 2.6 Jupyter Notebook基本使用（15分钟）

**Jupyter Notebook**是数据工程师的"实验草稿纸"，适合：
- 数据探索（边写代码边看结果）
- 教学演示
- 数据分析报告

**启动**：

```bash
# 在项目目录下
jupyter notebook
# 或
jupyter lab
```

**基本操作**：
- `Shift + Enter`：运行当前单元格
- `Ctrl + Enter`：运行当前单元格但不跳转
- `Esc + A`：在上方插入单元格
- `Esc + B`：在下方插入单元格
- `Esc + M`：切换为Markdown模式
- `Esc + Y`：切换为Code模式

### 练习2：编写完整的环境信息脚本（20分钟）

```python
"""
环境诊断脚本：完整输出当前开发环境的所有关键信息
用于在GitHub仓库中记录环境配置
"""
import sys
import os
import platform
from datetime import datetime

def get_python_info():
    return {
        "version": sys.version,
        "executable": sys.executable,
        "implementation": platform.python_implementation(),
        "compiler": platform.python_compiler(),
    }

def get_os_info():
    return {
        "system": platform.system(),
        "release": platform.release(),
        "version": platform.version(),
        "machine": platform.machine(),
        "processor": platform.processor(),
    }

def get_env_vars():
    relevant_vars = ["PATH", "CONDA_DEFAULT_ENV", "HOME", "USERPROFILE"]
    return {k: os.environ.get(k, "N/A") for k in relevant_vars}

def get_installed_packages():
    packages = {}
    try:
        import numpy; packages["numpy"] = numpy.__version__
    except ImportError:
        packages["numpy"] = "未安装"
    try:
        import pandas; packages["pandas"] = pandas.__version__
    except ImportError:
        packages["pandas"] = "未安装"
    try:
        import matplotlib; packages["matplotlib"] = matplotlib.__version__
    except ImportError:
        packages["matplotlib"] = "未安装"
    return packages

def main():
    report = []
    report.append("=" * 60)
    report.append("       大数据学习之旅 —— 环境诊断报告")
    report.append(f"       生成时间: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
    report.append("=" * 60)

    report.append("\n【Python信息】")
    for k, v in get_python_info().items():
        report.append(f"  {k}: {v}")

    report.append("\n【操作系统信息】")
    for k, v in get_os_info().items():
        report.append(f"  {k}: {v}")

    report.append("\n【环境变量】")
    for k, v in get_env_vars().items():
        report.append(f"  {k}: {v}")

    report.append("\n【已安装的核心包】")
    for k, v in get_installed_packages().items():
        report.append(f"  {k}: {v}")

    report.append("\n" + "=" * 60)

    report_text = "\n".join(report)
    print(report_text)

    # 同时保存到文件
    with open("env_report.txt", "w", encoding="utf-8") as f:
        f.write(report_text)
    print("\n报告已保存到 env_report.txt")

if __name__ == "__main__":
    main()
```
