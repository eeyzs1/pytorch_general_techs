```bash

find . -name "*.py" -exec cat {} + | grep -cv '^\s*$\|^\s*#'

# 统计提交次数

find . -name "*.py" | wc -l
```

## 七、常见问题

**Q: 进度落后了怎么办？**

调整策略：
1. 识别瓶颈：是概念不理解？还是练习做不完？还是时间不够？
2. 调整节奏：可以适当跳过选做内容，先保证核心进度
3. 求助：在群里提问，找同学讨论，预约答疑时间
4. 不要焦虑：学习不是竞赛，理解比速度重要

**Q: 技能树怎么判断自己到了哪个等级？**

自测方法：
- ⭐ 了解：能向别人解释这个概念（费曼学习法）
- ⭐⭐ 掌握：能在项目中独立使用，不需要查文档
- ⭐⭐⭐ 精通：能教别人，能解决别人解决不了的问题，能优化改进

**Q: 每周学习时间不够3.5小时怎么办？**

最低保障：
- 每天至少1小时编码时间
- 周末至少3小时集中学习
- 宁可每天少学一点，也不要一周不学然后突击
- 持续性 > 强度

**Q: 如何避免"学了就忘"？**

对抗遗忘：
1. 每周复习：周六花1小时回顾本周内容
2. 项目驱动：在项目中使用学过的知识
3. 教别人：技术分享是最好的复习
4. 记笔记：用自己的话写，不是抄文档
5. 间隔重复：第1天、第3天、第7天、第30天各复习一次
